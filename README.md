# ArtificialLife

1. Get a window and draw shapes — SDL2 or a raw framebuffer, just render circles and lines
2. Build a simple physics sim — nodes connected by springs, Verlet integration, gravity, ground collision
3. Define a creature as data — nodes, connections, muscle timings encoded as a genome
4. Animate one creature — hardcode a genome and watch it move
5. Write the fitness function — measure horizontal distance traveled
6. Add random creature generation — spawn a population of random genomes
7. Add selection and mutation — keep the best, mutate copies, repeat
8. Watch it evolve
